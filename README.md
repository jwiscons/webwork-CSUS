# webwork-CSUS
Problem banks for use with [WeBWorK](http://webwork.maa.org). This repository serves as a workspace for problem development at [California State University, Sacramento](https://www.csus.edu). 

## Overview
The following problem sets are currently being developed/maintained here:
* Calculus 1 (the "Scott Problems")
* Mathematical Reasoning topics including:
   - voting theory
   - graph theory
   - sequences
